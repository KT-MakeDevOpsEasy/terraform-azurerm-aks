# Changelog

## v1.0.0

### Features

- AKS cluster with Azure CNI networking
- User-assigned managed identity with Network Contributor role
- System node pool with critical-addons-only taint and availability zones
- Dynamic user node pools via map variable
- Calico network policy support (configurable)
- Azure AD RBAC integration
- OIDC issuer and workload identity federation
- Key Vault secrets provider with auto-rotation
- Container Insights via OMS agent
- Cluster autoscaler with tunable profile
- API server access profiles (authorized IPs or private cluster)
